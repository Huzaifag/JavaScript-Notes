## Lesson 1.7 Hoisting:
Variables and functions can be hoisted which means that their declaration can be moved on the top of the code.

Example : 
```javascript
// use test variable before declaring
console.log(test);

// declare and initialize test variable
var test = 5;

// Output: undefined
```
